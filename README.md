# Promo Adictos — MVP Automatizado

Automatiza la detección y publicación de ofertas de Mercado Libre hacia Telegram.

## Instrucciones rápidas
1. Coloca tu archivo `.env` dentro de esta carpeta.
2. Instala dependencias:
   python -m venv .venv
   .venv\Scripts\activate
   pip install -r requirements.txt
3. Ejecuta:
   python -m src.main
